# Table

Table, `<table>`, and the related `<tr>`, `<th>`, `<td>`, `<thead>`, and `<tbody>` tags, are used to display tabular data

## Available Classes

Styling can be applied in the following ways:

* `.base--table` - class
* `.base--STYLED table` - `<table>` tag inside a class of `.base--STYLED`
